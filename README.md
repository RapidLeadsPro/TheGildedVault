# The Gilded Vault (梁金藏石)

Singapore's first AI-powered Bazi reading and premium crystal house. Production site for [thegildedvault.sg](https://thegildedvault.sg).

## Stack

- **Framework**: Next.js 15 (App Router) + React 19
- **Language**: TypeScript (strict)
- **Styling**: Tailwind CSS 3.4 + custom CSS tokens
- **Typography**: Cormorant Garamond (display) + Inter (sans) + Noto Serif SC (Hanzi)
- **Hosting**: Vercel
- **Domain**: thegildedvault.sg

## Pages live at launch

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm start
```

## Deployment

Auto-deploys to Vercel on push to `main`. Preview deploys on every PR.

```bash
vercel              # preview
vercel --prod       # production
```

## SEO

- Per-page `<title>` + `<meta description>` from the SEO strategy doc
- Open Graph + Twitter card metadata
- LocalBusiness JSON-LD on every page
- `sitemap.xml` and `robots.txt` auto-generated by Next.js
- Mobile-first responsive layout
- `<3s` LCP target via next/image + next/font
